# Restaurant Management System (DSA)
A comprehensive management application focusing on Data Structures and File Handling.

## 🚀 Key Features
* **Linked List Management:** Dynamically handles menu items and feedback nodes.
* **Order Queue:** Manages customer orders using Front and Rear pointers.
* **Persistent Storage:** Saves all data to `.txt` files so it remains after the program closes.
* **Admin/Customer Panels:** Separate login and functionalities for staff and guests.

## 🛠️ Concepts Used
* **Data Structures:** Linked Lists, Queues.
* **OOP:** Classes, Objects, and Encapsulation.
* **I/O:** File Handling with `fstream'.
